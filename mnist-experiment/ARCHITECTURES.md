## 3. ResNet — MiniResNet

```
┌──────────────────────────────────────────────────────────────────────┐
│                         MiniResNet                                   │
├──────────────────────────────────────────────────────────────────────┤
│                                                                       │
│                          INPUT (1×28×28)                              │
│                              │                                        │
│                              ▼                                        │
│  ┌─────────────────────────────────────────┐                        │

│                       │                                             │
│                       ▼                                             │
│  ┌─────────────────────────────────────────┐                        │
│  │          RESIDUAL BLOCK 1 (32→32)       │                        │
│  │  ┌─────────────────────────────────┐    │                        │
│  │  │ Conv3×3(32→32) + BN + ReLU      │    │                        │
│  │  │ Conv3×3(32→32) + BN             │    │                        │
│  │  │         +                       │    │                        │
│  │  │   (shortcut: identity)          │    │                        │
│  │  │         ↓                       │    │                        │
│  │  │      ReLU(output)               │    │                        │
│  │  └─────────────────────────────────┘    │                        │
│  └────────────────────┬────────────────────┘                        │
│                       │                                             │
│                       ▼ MaxPool2d(2×2)                               │
│                                                                       │
│  ┌─────────────────────────────────────────┐                        │
│  │          RESIDUAL BLOCK 2 (32→64, stride=2)                      │
│  │  ┌─────────────────────────────────┐    │                        │
│  │  │ Conv3×3(32→64, stride=2)        │────┼──→ shortcut (1×1 conv) │
│  │  │ Conv3×3(64→64) + BN             │    │                        │
│  │  └─────────────────────────────────┘    │                        │
│  └────────────────────┬────────────────────┘                        │
│                       │                                             │
│                       ▼                                             │
│  ┌─────────────────────────────────────────┐                        │
│  │          RESIDUAL BLOCK 3 (64→64)       │                        │

│  └────────────────────┬────────────────────┘                        │
│                       │                                             │
│                       ▼ MaxPool2d(2×2)                               │
│                                                                       │
│  ┌─────────────────────────────────────────┐                        │
│  │          RESIDUAL BLOCK 4 (64→128, stride=2)                      │
│  │  ┌─────────────────────────────────┐    │                        │

│  │  └─────────────────────────────────┘    │                        │
│  └────────────────────┬────────────────────┘                        │

│                       ▼                                             │

## 4. ViT — MiniViT

```
┌──────────────────────────────────────────────────────────────────────┐
│                           MiniViT                                     │
├──────────────────────────────────────────────────────────────────────┤
│                                                                       │
│   INPUT (1×28×28)                                                     │
│       │                                                               │
│       ▼                                                               │
│   ┌────────────────────────────────────────────────────────────────┐ │
│   │              PATCH EMBEDDING (7×7 patches, stride=7)           │ │
│   │                                                                │ │
│   │   ┌────┬────┬────┬────┐                                       │ │
│   │   │ P0 │ P1 │ P2 │ P3 │   Conv2d(1→64, kernel=7, stride=7)   │ │
│   │   ├────┼────┼────┼────┤   → 16 patches (4×4 grid)            │ │
│   │   │ P4 │ P5 │ P6 │ P7 │   → Each patch embedded to 64-dim    │ │
│   │   ├────┼────┼────┼────┤                                       │ │
│   │   │P8  │P9  │P10 │P11 │                                       │ │
│   │   ├────┼────┼────┼────┤                                       │ │
│   │   │P12 │P13 │P14 │P15 │                                       │ │
│   │   └────┴────┴────┴────┘                                       │ │
│   └────────────────────┬───────────────────────────────────────────┘ │
│                        │                                             │
│                        ▼                                             │
│   ┌────────────────────────────────────────────────────────────────┐  │
│   │              ADD POSITIONAL EMBEDDING (learned)                │  │
│   │   [patch_0, patch_1, ..., patch_15] + [pos_0, pos_1, ..., pos_15]│  │
│   └────────────────────┬───────────────────────────────────────────┘  │
│                        │                                             │
│                        ▼                                             │
│   ┌────────────────────────────────────────────────────────────────┐  │
│   │                    TRANSFORMER BLOCK × 2                       │  │
│   │                                                                 │  │
│   │   LayerNorm → MultiHeadAttention(4 heads, d_k=16)            │  │
│   │        ↓                                                      │  │
│   │   LayerNorm → MLP(GELU)                                       │  │
│   │                                                                 │  │
│   └────────────────────┬───────────────────────────────────────────┘  │
│                        │                                             │
│                        ▼                                             │
│                  Mean Pooling (over 16 patches)                       │
│                        │                                             │
│                        ▼                                             │
│                  Linear(64→10) → class prediction                    │
│                                                                       │
│   Parameters: ~40K                                                   │
│   Key Feature: Self-attention captures global relationships          │

## 5. CapsuleNetwork

```
┌──────────────────────────────────────────────────────────────────────┐

├──────────────────────────────────────────────────────────────────────┤

│       ▼                                                               │

│                        │                                             │
│                        ▼                                             │
│   ┌─────────────────────────────────────┐                            │

│   │  Conv2d(256→256×8, 9×9, stride=2)  │                            │
│   │  → reshaped to [batch, 8 caps,     │                            │
│   │              8×8 spatial, 8 dim]    │                            │
│   └────────────────────┬────────────────┘                            │
│                        │                                             │
│                        ▼                                             │
│   ┌─────────────────────────────────────┐                            │
│   │         CAPSULE LAYER 1              │                            │
│   │  8 capsules → 8 capsules, dim=8    │                            │
│   │  Dynamic routing (iterations=3)     │                            │
│   └────────────────────┬────────────────┘                            │
│                        │                                             │
│                        ▼                                             │
│   ┌─────────────────────────────────────┐                            │
│   │         CAPSULE LAYER 2 (FINAL)      │                            │
│   │  8×8×8=512 → 10 capsules, dim=16     │                            │
│   │  Dynamic routing to 10 digit classes │                            │
│   └────────────────────┬────────────────┘                            │
│                        │                                             │
│                        ▼                                             │
│            Length of each capsule = class score                       │
│                        │                                             │
│                        ▼                                             │
│               [10 class scores] → argmax                             │
│                                                                       │
│   Decoder (reconstruction path, shown for completeness):             │
│   ┌──────────────────────────────────────────────────────────────┐  │
│   │  Capsule(16×10) → Linear(160→512) → ReLU                     │  │
│   │                  → Linear(512→1024) → ReLU                    │  │
│   │                  → Linear(1024→784) → Sigmoid                │  │
│   │                  → 28×28 reconstruction                       │  │
│   └──────────────────────────────────────────────────────────────┘  │
│                                                                       │
│   Parameters: ~320K                                                   │
│   Key Feature: Vector-valued capsules preserve spatial info          │

## 6. TinyGNN — Graph Neural Network

```
┌──────────────────────────────────────────────────────────────────────┐
│                          TinyGNN                                      │
├──────────────────────────────────────────────────────────────────────┤
│                                                                       │
│   INPUT (1×28×28)                                                     │
│       │                                                               │
│       ▼                                                               │
│   ┌─────────────────────────────────────┐                            │
│   │  Unfold into 9 patches (3×3 grid)   │                            │
│   │  Kernel: 9×9, Stride: 9             │                            │
│   │  Each patch = 81 features           │                            │
│   └────────────────────┬────────────────┘                            │
│                        │                                             │
│                        ▼                                             │
│   ┌─────────────────────────────────────┐                            │
│   │       NODE EMBEDDING                 │                            │
│   │  Linear(81→64) per node             │                            │
│   │  9 nodes total                      │                            │
│   └────────────────────┬────────────────┘                            │
│                        │                                             │
│                        ▼                                             │
│   ┌─────────────────────────────────────┐                            │
│   │       MESSAGE PASSING LAYER × 2     │                            │
│   │                                      │                            │
│   │  For each node:                     │                            │
│   │    1. Gather neighbor features      │                            │
│   │    2. Compute messages:             │                            │
│   │       Linear(64×2→64) → ReLU        │                            │
│   │       Linear(64→64)                  │                            │
│   │    3. Aggregate (mean)              │                            │
│   │    4. Update: concat(nodes, msg)    │                            │
│   │       Linear(128→64) → ReLU          │                            │
│   │    5. Residual: nodes += updated    │                            │
│   └────────────────────┬────────────────┘                            │
│                        │                                             │
│   ┌────────────────────┴────────────────┐                            │
│   │         3×3 GRID CONNECTIVITY        │                            │
│   │                                      │                            │
│   │       (0)──(1)──(2)                  │                            │
│   │        │  ╲│  ╱ │                    │                            │
│   │       (3)──(4)──(5)                  │                            │
